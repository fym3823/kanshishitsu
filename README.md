# 統合監視室勤務者として常識的で文化的な最低限度の有するべき知識

電気主任技術者（第三種・第二種）試験に向けた、○×・選択式のクイズ学習サイトです。
1問ずつ気軽に解き、その場で正誤判定と解説が出ます。まずは **法規** のみ収録（他科目は今後追加）。

## ファイル構成

| ファイル | 役割 |
|---|---|
| `index.html` | クイズ画面（トップページ） |
| `style.css` | デザイン |
| `app.js` | 出題・判定・統計のしくみ |
| **`questions.js`** | **問題データ本体（＝これがバックアップです）** |

---

## 1. まず手元で動かす

`index.html` をブラウザにドラッグ＆ドロップ（またはダブルクリック）するだけで動きます。
サーバー不要・オフラインでも動作します。

---

## 2. 無料でインターネット公開する（GitHub Pages）

ランニングコスト **0円** で誰でもアクセスできる形にできます。

1. GitHub の無料アカウントを作成（https://github.com）
2. 新しいリポジトリを作成（例：`kanshishitsu`／Public）
3. このフォルダ内の全ファイルをアップロード（またはgitでpush）
4. リポジトリの **Settings → Pages** を開く
5. 「Build and deployment」の Source を **Deploy from a branch**、Branch を **main / (root)** に設定して Save
6. 数十秒後、`https://ユーザー名.github.io/kanshishitsu/` で公開されます

以降は `questions.js` を差し替えて push するだけで、サイトが自動更新されます。

---

## 3. 問題を追加・更新する（Claude一括生成方式）

問題は Claude にまとめて作成を依頼し、**`questions.js` を丸ごと差し替えて取り込みます。**

1. Claude に「この形式で法規の問題を◯問作って」と依頼する（形式は下記／`questions.js` の先頭にも見本あり）
2. 受け取った新しい `questions.js` で、既存の `questions.js` を上書き保存する
3. 公開（push）する → サイトが更新される

`id` は他と重複しない管理番号にします（例：`houki-0021`, `houki-0022` …）。

### 書き方の見本

○×問題:
    {
      id: "houki-0021",
      subject: "法規",
      topic: "電圧の区分",
      type: "ox",
      question: "問題文をここに。",
      answer: true,            // true=○(正しい) / false=×(誤り)
      explanation: "解説をここに。"
    },

選択問題（2〜3択）:
    {
      id: "houki-0022",
      subject: "法規",
      topic: "接地工事",
      type: "choice",
      question: "問題文をここに。",
      choices: ["選択肢1", "選択肢2", "選択肢3"],
      answer: 0,               // 正解の番号（0=1つ目, 1=2つ目, 2=3つ目）
      explanation: "解説をここに。"
    },

## 4. バックアップの考え方（重要）

**`questions.js` そのものが、出題データであり、同時にバックアップです。**

- 手元のPCで `questions.js` を差し替え → 公開（push）すると、**サイト更新とPC上のバックアップ保存が同時に成立**します。
- GitHubにpushすれば、GitHub側にも履歴つきで全問題が保存されます（過去のどの版にも戻せます）。
- サイトからダウンロードする必要はありません。「差し替えるファイル＝保存されるファイル」です。

念のためもう一段安心したい場合は、`questions.js` を定期的に別フォルダやクラウド（OneDrive等）へコピーしておけば二重のバックアップになります。

## 5. 収録内容と精度について

初期収録（法規 20問）は、電圧区分・主任技術者・保安規程・電気工事士法・接地工事・絶縁抵抗・需要率／負荷率／不等率・風圧荷重など、頻出の基礎を過去問の枠にとらわれず広く配置しています。

法令・技術基準は改正されることがあります。出題を追加・見直しする際は、最新の電気事業法／電気設備技術基準（電技・解釈）で数値や定義をご確認ください。
